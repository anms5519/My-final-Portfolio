Markdown Cheatsheet

×

Headers

# Heading 1
Creates a top-level heading
## Heading 2
Creates a second-level heading
### Heading 3
Creates a third-level heading

Styling Text

**Bold Text**
Makes text bold
*Italic Text*
Makes text italic
~~Strikethrough~~
Strikes through text

Lists

- Item 1
- Item 2
- Item 3
Creates an unordered list
1. First item
2. Second item
3. Third item
Creates an ordered list

Links and Images

[Link Text](https://example.com)
Creates a hyperlink
![Alt Text](image-url.jpg)
Embeds an image

Blockquotes and Code

> This is a blockquote
Creates a blockquote
`inline code`
Creates inline code
```
code block
```
Creates a code block

Tables

| Header 1 | Header 2 |
| -------- | -------- |
| Cell 1 | Cell 2 |
| Cell 3 | Cell 4 |
Creates a table

Miscellaneous

---
Creates a horizontal rule
\*escaped\*
Escapes markdown characters
