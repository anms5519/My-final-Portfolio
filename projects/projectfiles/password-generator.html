Password Security Tips

  • Use a different password for each of your important accounts
  • Use a password manager to store and autofill your passwords securely
  • Enable two-factor authentication (2FA) whenever possible for additional security
  • Longer passwords are generally more secure than complex ones
  • Avoid using easily guessable information (birthdays, names, etc.) in your passwords
